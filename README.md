## Общая информация
Здесь будет собраны мои конспекты по различным темам: от вышмата и линейной алгебры до программирования
Здесь, конечно, будут не все темы, но они будут периодически обновляться (надеюсь)
### Формат конспектов
Конспекты будут предсталены в PDF-формате. Данный формат универсален и его можно прочитать на многих гаджетах.
Здесь также будут присутствовать конспекты по другим предметам, которые есть у меня в ВУЗе, например, безопасность жизнедеятельности (БЖД)

Конспекты по профильным дисциплинам (математика, физика, программирование и т.п.) я стараюсь совмещать с практикой, чтобы материал усваивался лучше.

_Коспекты по лекциям_ помечены хэштегом - #лекция.
_Обычные коспекты_ помечены хэштегом - #конспекты.

### Как читать название конспектов лекций:
Первые буквы означают сокращенное название предмета (И - информатика; П - программирование; БЖД - безопасность жизнд.-ти и т.д.);
Следующая буква с числом означает, что за тип занятия (Л - лекция, С - семинар), а число - номер занятия;
В конце идет дата проведения лекции (семинара).

Пример: БЖД-Л9-10.11.2025
БЖД - Безопасность жизнедеятельности;
Л9 - девятая лекция;
10.11.2025 - дата проведения лекции
