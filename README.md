## Общая информация
Здесь будет собраны мои конспекты по различным темам: от вышмата и линейной алгебры до программирования
Здесь, конечно, будут не все темы, но они будут периодически обновляться (надеюсь)
### Формат конспектов
Конспекты будут предсталены в PDF-формате. Данный формат универсален и его можно прочитать на многих гаджетах.
Здесь также будут присутствовать конспекты по другим предметам, которые есть у меня в ВУЗе, например, безопасность жизнедеятельности (БЖД)

Конспекты по профильным дисциплинам (математика, фижика, программирование и т.п.) я стараюсь совмещать с практикой, чтобы материал усваивался лучше.
### Где я веду конспекты
Конспекты ведутся в программе Obsidian: https://obsidian.md/

Установленные плагины:
- Excalidraw

Также я использую сайт Mathcha для записи сложных или больших формул, а также для черчения графиков: https://www.mathcha.io/editor
